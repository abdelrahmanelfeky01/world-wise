# 🌍 World Wise — Track Countries You've Visited

A travel tracking app that lets you log every city you've visited by clicking on an interactive world map, with personal notes and dates for each trip.

## ✨ Features
- 🗺️ Click on any city on the map to log a visit
- 📝 Add personal notes and travel dates for each city
- 🌐 Auto-fetches city and country name from coordinates (reverse geocoding)
- 🗂️ View all visited cities and countries in a sidebar list
- 🔐 Fake authentication with protected routes using React Router
